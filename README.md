
# Maruli Asido | Software Developer Portfolio

## 🚀 How to Run Locally

This project uses **Vite** for a lightning-fast development experience.

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start the development server**:
    ```bash
    npx vite
    ```
3.  **Open the browser**:
    Navigate to `http://localhost:3000`.

## 🏗️ Architecture
- **Tech Stack**: React 19, Tailwind CSS, TypeScript.
- **Title**: Software Developer.
- **Focus**: HRIS & Financial Systems optimization.
